# نصب - اجرا مستندات

این برنامه همانند همه برنامه مبتنی بر Node با دستور npm install ماژول استفاده شده آن نصب می گردد

نکته

از اینترنت مناسب استفاده کنید – بهتر است از VPN یا حداقل از سرویس https://shecan.ir بهره مند شوید

  • سرویس تــحریم شکن https://shecan.ir – کاملا رایگان و قانونی می باشد - پایداری آن نیز در حد متوسط به بالاست
  • معمولا فرآیند دانلود بدون مشکل می باشد – ولی در صورت مشکل شاخه node_modules حذف کنید و نسخه 11 نود استفاده کنید . البته سطح - - دامین همیشه منظم بوده و این موضوعات جز کار frontend-developer محسوب می شود
