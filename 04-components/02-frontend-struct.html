# فولدر public

این فولدر برای فایل های استاتیک مربوط برنامه در نظر گرفته شده است اسامی مربوط به خوانا بوده و نیاز به توضیحات اضافه ندارد

  • AppData
  • bundle - فایل های تولید شده از روی برنامه React/TypeScript
  • fonts - فایل های مربوط به فونت که استاندارد های وب تبعیت می کنند
  • icon - آیکون ها
  • img- تصاویر

# فولدر src\domain\frontend

فایل های مربوط به توسعه Frontend که در اغلب فایل ها از استاندارد React/Typescript خروجی این فولدر بوسیله webpack به فایل های domain*.js تبدیل میشود

  • api - کد ارتباط با سرور در قالب interface و توابع
  • assets - فایل های عکس ک بوسیله webpack به داخل bundle می رود
  • components - کامپونت های که توسط خودتان نوشته شده و در داخل برنامه استفاده است
  • dashboard-widgets - ویجت های داشبود - شامل تردد ها در 24 ساعت اخیر - آمار کلی
  • dialogs - پنجره ها موجود در برنامه که بصورت دیالوگ باز می شوند
  • icons - آیکون ها
  • reports - گزارشات مربوط به برنامه
